# Searchable Event Drawer
Searchable and compact drawer for UnityEvent

Insprired by https://github.com/neon-age/Compact-Unity-Events
but using AdvancedDropdown!

## Features
- Single line
- Collpasable with animation
- Copy All/ Selected
- Searchable method
- Reorderable
